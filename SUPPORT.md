# Support

## General Questions and Usage Help

Use GitHub Issues for questions, bug reports, and feature requests.

Before opening a new issue:

- search existing issues first
- include the environment and component you are working with
- explain what you expected and what actually happened
- choose the closest issue template so the report arrives with enough context

## Security Issues

For vulnerabilities, suspected secret exposure, or anything that should not be discussed in public, use the private reporting path in [SECURITY.md](./SECURITY.md).

## Scope Reminder

PersonalAgent is an active pre-release project. Some areas are intentionally still moving, so please treat examples and interfaces as evolving unless the docs explicitly mark them stable.
