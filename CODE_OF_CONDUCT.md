# Code of Conduct

## Our Standard

PersonalAgent is built around restraint, clarity, and respect. We expect contributors and maintainers to make this repository a place where people can collaborate without harassment, hostility, or performative aggression.

## Expected Behavior

- Be direct, but stay civil.
- Critique ideas, code, and decisions without attacking people.
- Assume good faith until behavior proves otherwise.
- Give clear technical feedback with enough context to be actionable.
- Respect different levels of experience, identities, and communication styles.

## Unacceptable Behavior

- Harassment, threats, intimidation, or sustained hostility.
- Personal insults, discriminatory language, or sexualized language or imagery.
- Doxxing, publishing private information, or encouraging dogpiles.
- Deliberately derailing discussions or repeatedly ignoring moderator direction.
- Bad-faith contributions intended to waste reviewer or maintainer time.

## Scope

This Code of Conduct applies in repository discussions, issues, pull requests, code review, and any official project communication channels.

## Reporting

If you experience or witness behavior that violates this Code of Conduct, report it privately at `github@abhisheksingh.com`.

## Enforcement

Maintainers may remove comments, close issues or pull requests, reject contributions, or restrict participation when behavior violates this Code of Conduct. Enforcement decisions are made to protect the health of the project and the people participating in it.
